# CC SS Contratistas e Inversiones E.I.R.L

Este repositorio contiene el sitio web para CC SS Contratistas e Inversiones E.I.R.L. 

## Estructura del Proyecto

- `index.html`: El archivo HTML principal.
- `estilos.css`: El archivo CSS con los estilos del sitio web.
- `video de fondo.mp4`, `logo.png`, `otro-video.mp4`, `red de salud.jpg`, `megantoni.jpg`, `kumpiru.jpg`, `casa sam.jpeg`, `casa sam 2.jpeg`, `equipo1.jpg`, `equipo2.jpg`: Archivos multimedia.

## Descripción

El sitio web presenta información sobre la empresa CC SS Contratistas e Inversiones E.I.R.L, incluyendo secciones sobre sus servicios, historia, y formas de contacto.

## Cómo Publicar

1. Clona este repositorio en tu máquina local.
2. Añade tus archivos multimedia a las carpetas adecuadas.
3. Realiza un `commit` de tus cambios y `push` a GitHub.
4. Habilita GitHub Pages en la configuración del repositorio.
